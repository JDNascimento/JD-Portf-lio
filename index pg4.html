Projetos Concluídos



💆‍♀️ Centro Estético Mariane Miranda


Este projeto foi desenvolvido com foco exclusivo na gestão interna do centro estético, atendendo às necessidades específicas da proprietária. O sistema permite o cadastro de clientes, agendamento de serviços personalizados, controle de fluxo de caixa, entre outras funcionalidades voltadas à organização administrativa do negócio. Construído inteiramente com HTML, CSS e JavaScript, o site oferece uma interface intuitiva, sem depender de frameworks ou banco de dados externos, garantindo leveza e praticidade. Foi um excelente exercício de front-end com lógica aplicada à rotina de pequenos negócios.


⚡ Calculadora de Consumo de Energia para Projetos Sociais


Desenvolvido com o objetivo de conscientizar comunidades sobre o consumo de energia elétrica, este site simula o uso diário de aparelhos e apresenta ao usuário dicas práticas de economia. A ferramenta calcula automaticamente o custo mensal com base em parâmetros simples, tornando o conteúdo acessível a todos. Criado com HTML, CSS e JavaScript, o projeto foi idealizado como suporte a ações sociais e educativas, incentivando o uso responsável de energia e contribuindo para uma mudança de comportamento ambientalmente consciente.


🎨 Portfólio de Artesanato Premiado – Prêmio Aldir Blanc


Este site foi desenvolvido para valorizar e divulgar o trabalho artesanal de uma artesã premiada, cujo talento foi reconhecido com o Aldir Blanc, parte da Política Nacional de Fomento à Cultura. O portfólio apresenta de forma visual e atrativa as peças produzidas, técnicas utilizadas e a trajetória da artesã, fortalecendo sua presença digital e ampliando seu alcance. Produzido com HTML, CSS e JavaScript, o projeto alia estética artesanal e funcionalidade, contribuindo para a valorização da cultura local e dos saberes tradicionais..
